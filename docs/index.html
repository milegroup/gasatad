What is GASATaD?

GASATaD reads .csv or .xls/.xlsx files and allows users to perform different statistical tests across columns of data. It generates high quality plots and it can export them to .pdf, .svg, .png, .tiff or .eps files.

Free to Use

Free to download and use for any purpose

Open Source

Modify and improve it as you wish

Portable

Binaries are available for Microsoft Windows, debian-based GNU Linux platforms and Apple MacOS

Reliable

It is based on mature python libraries, such as numpy, scipy, pandas or matplotlib

Useful

It generates high quality plots suitable to be included in technical reports or scientific papers

MacOS installer v1.4

To install GASATaD in a MacOS system, open a terminal and paste the following text:

bash <(curl -fsSL https://github.com/milegroup/gasatad/raw/master/docs/packages/GASATaD_1_4.MacInstall.sh)

This will install the homebrew package manager (root access rights are required), and all the packages GASATaD depends on. The downloading and installation process will take some minutes and will need 900 MB of disk, approximately.


To uninstall the homebrew package manager and GASATaD, open a terminal and paste:

bash <(curl -fsSL https://github.com/milegroup/gasatad/raw/master/docs/packages/GASATaD.MacUninstall.sh)

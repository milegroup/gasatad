A 5-minutes intro to GASATaD

What is GASATaD?

GASATaD reads .csv or .xls/.xlsx files and allows users to perform different statistical tests across columns of data. It generates high quality plots and it can export them to .pdf, .svg, .png, .tiff or .eps files.

Free to Use

Free to download and use for any purpose

Open Source

Modify and improve it as you wish

Portable

Binaries are available for Microsoft Windows, debian-based GNU Linux platforms and Apple MacOS

Reliable

It is based on mature python libraries, such as numpy, scipy, pandas or matplotlib

Useful

It generates high quality plots suitable to be included in technical reports or scientific papers
